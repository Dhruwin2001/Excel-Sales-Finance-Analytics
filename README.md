# 📊 Superstore Sales Analysis (Excel Project)

## 📌 Overview

This project analyzes retail sales data using Microsoft Excel.
The goal is to extract meaningful business insights from raw transactional data using data cleaning, transformation, and visualization techniques.

---

## 🛠️ Tools Used

* Microsoft Excel
* Power Query (Data Cleaning & Transformation)
* Pivot Tables
* Slicers & Interactive Filters
* Conditional Formatting

---

## 📂 Dataset

* Sample Superstore dataset
* Contains information on orders, customers, products, sales, profit, and regions

---

## 📊 Key Analysis

### 🔹 1. Sales Overview

* Regional sales performance analysis
* Interactive slicers for **Region** and **Year**
* Quick comparison of sales across different regions

---

### 🔹 2. Customer Performance Analysis

* Year-wise customer sales breakdown
* Growth % calculation to track performance trends
* Identification of high-performing and declining customers

---

### 🔹 3. Product Analysis

* Top 10 products based on sales
* Bottom 10 products (low/negative performance)
* Profit analysis by product category
* Visualization using charts

---

## 📈 Key Insights

* Certain regions consistently outperform others in total sales
* Some customers show extreme growth trends (both positive and negative)
* A small number of products contribute significantly to total revenue
* Several products generate low or negative profit, indicating optimization opportunities
